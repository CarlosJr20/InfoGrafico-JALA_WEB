Introdução ao Desenvolvimento Web

Guia Básico para Iniciantes

HTML

HyperText Markup Language - A linguagem base para criar páginas web.

<h1>Título</h1>
<p>Parágrafo</p>

CSS

Cascading Style Sheets - Linguagem para estilizar páginas web.

button {
  color: blue;
  padding: 10px;
}

JavaScript

Linguagem de programação para criar interatividade em websites.

function sayHello() {
  alert("Olá!");
}

Navegador

Software que interpreta o código e exibe páginas web.

Exemplos: Chrome, Firefox, Edge, Safari

Frontend

Parte visual do site que o usuário interage diretamente.

Layout Design Interação

Backend

Parte do sistema que processa dados no servidor.

Servidor Banco de Dados API
